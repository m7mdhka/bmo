## BMO – Local Developer Workspace Platform

BMO is an open-source tool that lets anyone create and manage coding projects on their own computer, through a simple web interface.

Instead of installing programming tools and editors manually, users just:

- **Run a Docker image** on their machine.
- **Open the browser** to access BMO.
- **Create projects** (like small apps or websites) from ready-made templates, without needing to know how to use the terminal.

BMO is inspired by platforms like Replit and Lovable, but it runs **locally** and keeps all your code and data on **your own device**. It is organized as a single monorepo with:

- A **Python backend** that powers the logic and APIs.
- A **Next.js + React frontend** that provides the web interface.

The goal is to make it easy for non‑technical users to start building projects, while still giving developers a solid, extensible foundation they can customize and improve.
